## 前台权限

前台权限是通过用户的声望值实现的，借鉴 stackoverflow 的设计，规则如下

| 声望值 | 权限 |
| --- | --- |
| 15 | 给评论投票 |
| 15 | 给话题投票 |
| 200 | 编辑评论 |
| 200 | 编辑话题 |
| 3000 | 删除评论 |
| 5000 | 删除话题 |

用户的声望哪来呢？看下面

| 声望值 | 权限 |
| --- | --- |
| -2 | 点踩评论 |
| -2 | 点踩话题 |
| 5 | 话题被点赞 |
| 10 | 评论被点赞 |
